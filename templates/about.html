About Sentiment Analysis App

This web application performs sentiment analysis on text using a pre-trained LSTM model. Users can input a text, and the application will predict whether the sentiment is Negative, Neutral, or Positive.

How it Works

The application uses natural language processing techniques to preprocess the input text, tokenize it, and apply lemmatization. The processed text is then fed into a pre-trained LSTM model to make sentiment predictions.

Technologies Used

  • Flask: A web framework for Python
  • TensorFlow: An open-source machine learning library
  • NLTK: A natural language processing library
